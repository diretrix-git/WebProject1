What We Learned

Six core topics that took us from zero to building real web pages.

🏗️

HTML Basics

The skeleton of every webpage. We learned semantic elements, document structure, forms, and tables.

🎨

CSS Styling

Selectors, the box model, colors, and typography. We turned plain HTML into visually appealing pages.

📐

CSS Flexbox

One-dimensional layouts made easy. We built responsive navbars and card rows using flex containers.

�

CSS Grid

Two-dimensional layout power. We built complex page structures and responsive grids with ease.

⚙️

JavaScript DOM

Selecting and modifying HTML elements with JavaScript. We learned how the browser represents a page.

🖱️

JS Events

Making pages interactive. We handled clicks, form submissions, and keyboard input using event listeners.
